GIGAIPC H310AI BIOS F5 Firmware UPDATE

Open Firmware Folder View Update Instruction (PDF)

Instructions Summary

Step 1: Prepare a USB drive formatted to FAT32.

Step 2: Copy the BIOS file and flash tool to the USB root directory.

Step 3: Power on the device and enter BIOS using the [DEL] key.

Step 4: Navigate to the “Save & Exit” tab and select “Launch EFI Shell from filesystem device.”

Step 5: Run the update command: fs0:\flash.bat

Step 6: Wait for the update process to complete. The system will automatically reboot.

⚠ Please do not power off the system during the update process.
